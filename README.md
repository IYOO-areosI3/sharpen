В· CatвҖ‘style EVERYTHING вҖ“ from the prompt (в•ӯв”Җв”Җ :3 SharpenOS) to error messages (:3 NyДҒ~), itвҖҷs all pawsitively adorable.

рҹ“ҡ Command Reference

ed вҖ“ Enter Directory

Replaces cd (which is strictly forbidden here!)

```
ed PATH:<directory>
```

Examples:

```
ed PATH:/sdcard/Documents
ed PATH:../projects
ed PATH:          # stays in current directory
```

Tip: always start the argument with PATH: вҖ” itвҖҷs part of the cutieвҖ‘cat charm!

---

lp вҖ“ List Path

Replaces ls

```
lp [options]
```

Options:

В· -nofile вҖ“ show only directories
В· -nofolder вҖ“ show only files
В· -ext .ext вҖ“ filter by file extension (e.g. -ext .lua)

Colors:

В· Blue вҶ’ directory
В· Cyan вҶ’ symbolic link
В· Green вҶ’ executable
В· Magenta вҶ’ socket / pipe
В· Default вҶ’ regular file

Examples:

```
lp
lp -nofile
lp -ext .txt
```

---

file вҖ“ Create a New File

Replaces touch

```
file <filename>
```

Examples:

```
file meow.txt
file purr-logs/scratch.log
```

---

directory вҖ“ Create a New Directory

Replaces mkdir

```

```

Examples:

```

directory ~/cat_projects/
```

---

ram вҖ“ Show Storage Usage

See disk usage with a cute bar :3

```
ram [path]
```

If no path is given, the current directory is used.

Output:

В· Colored usage bar (red = used, green = free)
В· Percentage used
В· HumanвҖ‘readable free / used / total sizes

Examples:

```

ram /sdcard
```
